## issuable

manage your github issues with ease!

[Try the Demo](https://routable-euojmyvlju.now.sh)

### quickstart

```
git clone https://github.com/sheminusminus/issuable
cd issuable
yarn
yarn start
```

### run in dev mode

`yarn start`

### run tests

`yarn test`

### build for production

`yarn build`
